# Time Field Model (TFM) for Galactic Dynamics: Eliminating Dark Matter

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

This repository contains code and data for **Paper #13** in the TFM Series:  
*"Eliminating Dark Matter: Wave Geometry in the Time Field Model as an Alternative for Galactic Dynamics"*.

## Overview
- **Key Contribution**: Derives spacetime wave parameters (λ, β) to explain galactic rotation curves without dark matter.
- **Features**:
  - TFM velocity profile calculation (`v_tfm(r)`).
  - χ² comparison between TFM and ΛCDM (NFW profiles).
  - HPC-compatible PDE solver for time-wave fields (`T⁺`, `T⁻`).
